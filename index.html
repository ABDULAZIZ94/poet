What is Poet?

Poet is a blog generator in node.js to generate routing, render markdown posts, and get a blog up and running fast. Poet may not make you blog-famous, and it may give you one less excuse for not having a blog, but, just imagine the insane hipster cred you get for having a blog in node.js. "Cool blog, is this Wordpress or something?" your square friend asks. "Nah dude, this is in node," you respond, while skateboarding off into the sunset, doing mad flips and stuff. Little do they know you just used Poet's autoroute generation to get your content up in that internet.

Getting Started

First thing first, throw Poet into your express app's package.json, or just install it locally with:

npm install poet

Once you have the Poet module in your app, just call it and pass in your express server and use some autorouting methods. Now you just need to make some posts and you're good to go!

		var
		  express = require('express'),
		  poet    = require('poet'),
		  app     = express.createServer();

		poet( app )
		  .createPostRoute()
		  .createPageRoute()
		  .createTagRoute()
		  .createCategoryRoute()
		  .init();

You can also pass in some configuration options, and specify a callback in the init method once all your posts are loaded up. Below are the config defaults:

	var
		express = require('express'),
		poet    = require('poet'),
		app     = express.createServer();

	poet( app, {
		posts: './_posts/',  // Directory of posts
		postsPerPage: 5,     // How many posts per page if you use pagination
		metaFormat: 'json'   // Which formatter should be used in posts
	})
		.createPostRoute()
		.createPageRoute()
		.createTagRoute()
		.createCategoryRoute()
		.init(function () {
			// Some callback to run once everything is set up
		});

Posts

Posts are constructed in markdown, prefixed by front matter via YAML or JSON. All attributes are stored in the posts object. An example of a blog post formatted with JSON Front Matter is below:

{{{
	"title" : "Hello World.js",
	"tags"  : [ "blog", "fun" ],
	"category" : "javascript",
	"date" : "8-9-2012"
}}}
